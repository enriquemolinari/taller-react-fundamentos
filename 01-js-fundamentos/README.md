# JavaScript Fundamentos

| Archivo | Descripción |
|---------|------------|
| **01-funciones.js** | Demuestra la declaración de funciones tradicionales, funciones flecha con return implícito y explícito, así como la sintaxis y diferencias entre cada tipo. |
| **02-objetos-arreglos.js** | Contiene ejemplos sobre la creación, manipulación y uso de objetos y arreglos en JavaScript, incluyendo métodos comunes y acceso a propiedades. |
| **03-iteraciones.js** | Muestra diversos métodos para iterar sobre arreglos como forEach, map, filter y otras técnicas de iteración en JavaScript. |
| **04-destructuring.js** | Explora la sintaxis de destructuring para extraer valores de objetos y arreglos de forma concisa, facilitando el acceso a propiedades anidadas. |
| **05-promesas.js** | Demuestra el uso de promesas, manejo de estados (pending, fulfilled, rejected) y cómo trabajar con código asincrónico en JavaScript. |

Para ejecutar alguno de los ejemplos, en una terminal:

```bash
cd 01-js-fundamentos
node 01-funciones.js
```
